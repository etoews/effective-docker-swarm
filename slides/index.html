Constraints

tl;dr You don't have access to the Docker Host

  • Cannot use volume mounts
    • Example --volume /some/host/path:/some/container/path
    • Except for --volume /var/run/docker.sock:/var/run/docker.sock
    • See Data section
  • Cannot use --privileged, --cap-add, or --cap-drop
  • AppArmor restrictions
  • Try not to docker rm the Carina containers

Service Discovery

Stack 1

  • etcd == distributed key value store
  • registrator == service registry bridge
  • confd == config writer + reloader
  • nginx/haproxy == load balancer

Stack 2

  • consul == distributed key value store
  • registrator == service registry bridge
  • consul-template == config writer + reloader
  • nginx/haproxy == load balancer

x node cluster

  • consul/etcd == x containers
  • registrator == x containers
  • consul-template/confd == sidecar process per load balancer
  • nginx/haproxy == container per service

Stack 3

  • interlock == service registry bridge + config writer + reloader
  • nginx/haproxy == load balancer

x node cluster

  • interlock == container per service
  • nginx/haproxy == container per service
